# ESP32-S3-DevBoard
Zadanie rekrutacyjne - projekt płytki PCB dla modułu ESP32-S3

Projekt kompletnej płytki rozwojowej opartej na układzie **ESP32-S3**, wykonany w ramach zadania rekrutacyjnego.

##  Kluczowe funkcjonalności
* **MCU:** ESP32-S3-WROOM (Wi-Fi + BLE)
* **Zasilanie i Komunikacja:** Port USB-C z układem LDO 3.3V.
* **Interakcja:** Przyciski BOOT i RESET, dioda LED statusowa.
* **Sensory:** Zintegrowany akcelerometr i żyroskop BMI270.

##  Struktura repozytorium
* `/Project_Files` - Pliki źródłowe KiCad (.kicad_sch, .kicad_pcb)
* `/Production` - Pliki Gerber i BOM przygotowane do zamówienia.
* `/step` - Model płytki w formacie .STEP.
